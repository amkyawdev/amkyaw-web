# AmkyawDev Website

Professional web development services with stunning 3D effects and modern design.

## Features

- 🎨 Modern UI/UX with 3D effects (Three.js)
- 📱 Fully responsive design
- 🔐 Firebase authentication
- 💬 Interactive chat widget
- 📊 Admin dashboard
- 🚀 Fast and optimized

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Deploy to Firebase
npm run deploy
```

## Tech Stack

- HTML5/CSS3
- JavaScript (Vanilla + Alpine.js)
- Three.js for 3D effects
- Firebase for authentication
- Express.js for API
